    Tools

    Tools in CAD Sketcher are either exposed as a workspacetool or as an operator. Note however that either of those use the same interaction system.

    Generic Tools

    View3D_OT_slvs_add_sketch

    Add a sketch

    States:
    - Workplane: Pick a workplane as base for the sketch. Types: [SlvsWorkplane]

    View3D_OT_slvs_delete_entity

    Delete Entity by index or based on the selection if index isn't provided

    View3D_OT_slvs_delete_constraint

    Delete constraint by type and index

    Workspacetools

    Workspacetools are used to interactively create entities. You can access them from the viewport's "T"-panel. Check the tools section to get familiar with the behavior of BGS tools.

    INFO: Interaction with addon geometry is only possible when one of the addon tools is active.

    Workspacetool Access Keymap

    Whenever one of the addon's tools is active the tool access keymap allows to quickly switch between the different tools.

    Key Modifier Action
    ESC - Activate Tool: Select
    P - Activate Tool: Add Point 2D
    L - Activate Tool: Add Line 2D
    C - Activate Tool: Add Circle
    A - Activate Tool: Add Arc
    R - Activate Tool: Add Rectangle
    S - Activate Tool: Add Sketch

    Constraints:

    Key Modifier Action
    Shift + D - Distance
    Shift + A - Angle
    Shift + O - Diameter
    Shift + C - Coincident
    Shift + V - Vertical
    Shift + H - Horizontal
    Shift + E - Equal
    Shift + P - Parallel
    Shift + N - Perpendicular
    Shift + T - Tangent
    Shift + M - Midpoint
    Shift + R - Ratio

    Basic Tool Keymap

    The basic tool interaction is consistent between tools.

    Key Modifier Action
    Tab - Jump to next tool state or property substate when in numerical edit
    0-9 / (-) - Activate numeric edit
    Enter / Lmb - Verify the operation
    Esc / Rmb - Cancel the operation

    While numeric edit is active

    Key Modifier Action
    Tab - Jump to next tool property substate
    0-9 - Activate numeric edit
    Minus(-) - Toggle between positive and negative values

    View3D_OT_slvs_select

    Select an entity

    Either the entity specified by the index property or the hovered index if the index property is not set

    Keymap:

    Key Modifier Action
    LMB - Toggle Select
    ESC - Deselect All

    INFO: LMB in empty space will also deselect all.

    View3D_OT_slvs_add_point3d

    Add a point in 3d space

    States:
    - Location: Set point's location.

    View3D_OT_slvs_add_line3d

    Add a line in 3d space

    States:
    - Startpoint: Pick or place line's starting point. Types: [SlvsPoint3D]
    - Endpoint: Pick or place line's ending point. Types: [SlvsPoint3D]

    View3D_OT_slvs_add_point2d

    Add a point to the active sketch

    States:
    - Coordinates: Set point's coordinates on the sketch.

    View3D_OT_slvs_add_line2d

    Add a line to the active sketch

    States:
    - Startpoint: Pick or place line's starting Point. Types: [SlvsPoint2D]
    - Endpoint: Pick or place line's ending Point. Types: [SlvsPoint2D]

    View3D_OT_slvs_add_circle2d

    Add a circle to the active sketch

    States:
    - Center: Pick or place circle's center point. Types: [SlvsPoint2D]
    - Radius: Set circle's radius.

    View3D_OT_slvs_add_arc2d

    Add an arc to the active sketch

    States:
    - Center: Pick or place center point. Types: [SlvsPoint2D]
    - Startpoint: Pick or place starting point. Types: [SlvsPoint2D]
    - Endpoint: Pick or place ending point. Types: [SlvsPoint2D]

    View3D_OT_slvs_add_rectangle

    Add a rectangle to the active sketch

    States:
    - Startpoint: Pick or place starting point. Types: [SlvsPoint2D]
    - Startpoint: Pick or place starting point. Types: [SlvsPoint2D]

    View3D_OT_slvs_add_workplane

    Add a workplane

    States:
    - Origin: Pick or place workplanes's origin. Types: [SlvsPoint3D]
    - Orientation: Set workplane's orientation.
